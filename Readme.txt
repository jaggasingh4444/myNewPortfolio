# Personal Portfolio Website

This is my personal portfolio website, created to showcase my skills, projects, certifications, and professional profile as an aspiring Cloud Engineer and IT professional.

## Live Website

🌐 **Website:** https://pandeyjagadish.com.np

## Project Overview

This portfolio website presents my background, technical skills, projects, resume, certifications, and contact information in a clean and responsive design.

The website was built using a Bootstrap-based template and customized according to my personal branding and career goals. It is deployed using **AWS Amplify** with GitHub integration for automatic deployment.

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Bootstrap
- GitHub
- AWS Amplify

## Cloud Deployment

This website is hosted on **AWS Amplify**. The GitHub repository is connected with AWS Amplify, so whenever changes are pushed to the main branch, AWS Amplify automatically builds and deploys the latest version of the website.

## Deployment Workflow

```text
GitHub Repository → AWS Amplify → Live Website
