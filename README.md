# Jagdish Pandey Personal Website

A lightweight static personal website built with HTML, CSS, and a small amount of JavaScript.

## Preview

Open `index.html` in a browser, or run a tiny local server from this folder:

```sh
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Customize

Update these areas first:

- `index.html`: name, intro copy, project cards, contact links, and email address.
- `styles.css`: color variables in `:root` if you want a different visual mood.
- `script.js`: small interactions for the mobile menu and reveal animations.
